# SendPulse

Todo application.

## Getting Started

Clone repo to your local machine:
```
git clone https://github.com/popelkin/sendpulse.git
```

Run docker-compose command:
```
docker-compose up -d
```

Open browser at:
```
http://sendpulse.loc/
```

## Running the tests

Run the tests from project directory:
```
php ./vendor/phpunit/phpunit/phpunit
```
